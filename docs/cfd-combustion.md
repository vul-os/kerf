# Reacting Flow and Combustion

> Simulate premixed and diffusion flames using Eddy Break-Up combustion with a single-step Arrhenius reaction — for furnace, gas turbine, and industrial burner analysis.

**Module**: `packages/kerf-cfd/src/kerf_cfd/combustion/reacting_flow.py`
**Shipped**: Wave 10
**LLM tools**: `cfd_combustion_run`

---

## What it is

Combustion CFD couples the flow equations with species transport and heat release. The Magnussen–Hjertager Eddy Break-Up (EBU) model is the standard engineering turbulent combustion model: the reaction rate is limited by the slower of chemical kinetics (Arrhenius) and turbulent mixing (proportional to ε/k), capturing the mixing-limited character of turbulent flames without solving detailed chemistry.

This module implements a 0D/1D combustion step (`step_combustion`) for testing and integration, plus the `magnussen_ebu_reaction_rate` function that computes the volumetric reaction rate for a premixed or diffusion flame. It supports CH₄/air, H₂/air, and user-defined fuel species. Engineers use it to estimate heat release rates and exit temperatures before running full 3D OpenFOAM reactingFoam cases.

## How to use it

### From chat (natural language)

> "Estimate the adiabatic flame temperature for methane-air combustion at equivalence ratio 1.0"

The LLM calls `cfd_combustion_run` with the fuel spec and equivalence ratio.

### From Python

```python
from kerf_cfd.combustion.reacting_flow import (
    FuelSpecies, CombustionMixture, step_combustion,
    magnussen_ebu_reaction_rate,
)

fuel = FuelSpecies(name="CH4", MW=16.04, LHV_J_kg=50e6)
mixture = CombustionMixture(
    fuel=fuel, oxidiser="air",
    T_K=300.0, P_Pa=101325.0, phi=1.0,
)
rate = magnussen_ebu_reaction_rate(mixture, k=0.1, eps=0.5)
print(f"Reaction rate: {rate:.3e} kg/(m³·s)")

state = step_combustion(mixture, dt=1e-4)
print(f"Temperature after 0.1ms: {state.T_K:.1f} K")
```

## How it works

The EBU reaction rate is: ω̇ = min(ω_Arr, A_EBU ρ (ε/k) min(Y_F, Y_O/s)) where ω_Arr is the Arrhenius rate, A_EBU is the Magnussen constant (4.0), Y_F and Y_O are fuel and oxidiser mass fractions, s is the stoichiometric oxygen-to-fuel ratio, and ε/k is the inverse turbulent time scale. The minimum of the two rates ensures both mixing and kinetics are rate-limiting.

Heat release is q = ω̇ × LHV (lower heating value), which appears as a source term in the energy equation.

## API reference

| Function | Returns | Purpose |
|---|---|---|
| `magnussen_ebu_reaction_rate(mixture, k, eps)` | `float` | Volumetric reaction rate kg/(m³·s) |
| `step_combustion(mixture, dt)` | `CombustionMixture` | Single time step integration |

`FuelSpecies` fields: `name`, `MW`, `LHV_J_kg`, `A_Arr`, `Ea_J_mol`.
`CombustionMixture` fields: `T_K`, `P_Pa`, `phi`, `Y_F`, `Y_O`, `heat_release_rate`.

## Example

```python
mix = CombustionMixture(FuelSpecies("H2", 2.016, 120e6), "air", 300, 101325, phi=0.8)
rate = magnussen_ebu_reaction_rate(mix, k=0.05, eps=0.3)
print(f"Lean H2 reaction rate: {rate:.2e} kg/(m³·s)")
```

## Honest caveats

The EBU model uses a single-step global reaction mechanism — it cannot predict NOx, CO, or soot formation, which require detailed multi-step chemistry (GRI-Mech 3.0). The model over-predicts reaction rates in near-stoichiometric mixtures; use a finite-rate/EDC model for accurate species predictions. The 0D step function is for testing; full 3D combustion runs use OpenFOAM `reactingFoam` generated by the OpenFOAM bridge.
