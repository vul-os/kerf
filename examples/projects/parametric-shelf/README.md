# Parametric Shelf

Modular shelving unit with adjustable width, depth, height, and shelf count.

## Files

- `params.equations` — Parameters: width (600mm), depth (250mm), height (800mm), shelf_count (4)
- `shelf.feature` — Pad + pocket + pattern_repeat feature tree
- `drawing.drawing` — A3 landscape: front + side views with dimensions

## Parameters

| Name | Default | Description |
|------|---------|-------------|
| width | 600 mm | Overall width |
| depth | 250 mm | Shelf depth |
| height | 800 mm | Total height |
| shelf_count | 4 | Number of shelves |

## Features

- Two side panels (full height)
- Repeated shelf pads (18mm thick)
- Mounting pockets on side panels
- Thin back panel for stability
